CSS Tag vs Class vs ID Selectors

Use class to target groups and ID when you want to be more specific. A particular ID can only be used once inside a page.

Tag Selector

img { background-color: red; }

Class Selectors

.spider{ background-color: #FF5152; }
.butterfly{ background-color: #6D9F3B; }
.snail { background-color: #F7DB1D;


Any html elements can have more than one class.

img class="snail circular"


.circular {border-radius: 100%}

ID Selectors

Ie. giving an ID to H1: h1 id="heading"

In CSS you would target it like dis: #heading { color: #6F2408;}
